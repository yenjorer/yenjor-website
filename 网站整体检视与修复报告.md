## 🛠️ 已修复问题

### 1. 手机版Banner标签透明度修复

**问题描述：**
- Hero区域的卖点标签在移动端使用 `bg-white/90`（90%透明度）
- 在某些背景下，透明背景可能导致文字不够清晰

**修复方案：**
- 为所有卖点标签添加专属类名 `hero-feature-tag`
- 新增移动端媒体查询CSS：
  ```css
  @media (max-width: 768px) {
      .hero-feature-tag {
          background: rgba(255, 255, 255, 0.98) !important;
          box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
          border-color: rgba(0, 0, 0, 0.08) !important;
      }
  }
  ```

**修复效果：**
- 移动端背景不透明度提升至 98%，文字更清晰
- 添加微妙阴影增强层次感
- 优化边框颜色提升可见度

**受影响的卖点标签：**
- Free Samples
- Fast Quotes
- Low MOQ
- 24hr Reply
- Good Prices
- Ship Worldwide
